Idetek development tools

On this website you will find multiple useful tools for the developer.

Available tools
JSON to excel and csv
Sometimes it is necessary to convert our json objects to a file type readable by people outside of the IT department. Idetek makes this job easy for you.
Password generator
Generates strong random passwords with uppercase, lowercase characters, numbers and symbols. We do not store the generated password. Everything is done on the client side.
Character counter
Analyze texts in a simple way. Count sentences, characters, words, paragraphs and spaces.
